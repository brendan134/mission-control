# Content Pipeline

## Content Types

### LinkedIn Post Template
```
Hook (attention-grabbing statement or question)
↓
Body (1-2 short paragraphs, value-focused)
↓
Call to Action (question to comments, or CTA)
```

### Newsletter Template
```
Subject: [Topic that solves a problem]

Quick tip: [One actionable insight]

Main content: [Dive deeper]

[Signature/CTA]
```

### Podcast Episode Structure
```
1. Intro (theme, guest context)
2. Main discussion (3-5 key points)
3. Takeaway/action item
4. Outro (next episode preview)
```

## Workflow

1. **Plan**: Add to content calendar
2. **Draft**: Use templates above
3. **Review**: Check against brand voice
4. **Schedule**: Queue for publishing
5. **Publish**: Post and track engagement

## Brand Voice
- Be direct, not salesy
- Focus on transformation, not features
- Ask questions to drive engagement
- Share real examples and results
