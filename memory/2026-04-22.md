### What We Tried

4. User deleted tunnel from Cloudflare dashboard
5. Created fresh tunnel via Cloudflare dashboard wizard (got new token)

### Current State
- cloudflared shows connections registering (4 connections)
- But tunnel shows "Inactive" in dashboard
- "Connection terminated" errors in logs
- "Tunnel not found" error when checking status

### DNS Status
- DNS record exists: mc.brendanrogers.au → tunnel
- But may be pointing to wrong tunnel ID after multiple recreations

### RESOLVED! ✅
- Tunnel now working
- Key fix: DNS Proxy Status must be set to **Proxied** (not DNS only)
- When Proxy Status = DNS only, traffic goes direct to IP, not through tunnel
- When Proxy Status = Proxied, Cloudflare routes through tunnel to origin

### Final Configuration
- Tunnel: Mission-Control (ID: 4a6b285e-882d-444f-bbe6-8f53dfff579b)
- DNS: mc.brendanrogers.au → Tunnel (Proxied)
- Ingress: mc.brendanrogers.au → http://localhost:3003
