## Lessons Learned (CRITICAL)

### What Went Wrong
1. **Code not pushed to GitHub** - Lost all today's work when snapshot restored
2. **Path confusion** - Write tool created literal folders when paths had special chars/wildcards (ISSION_*, UNFILTERED, etc.)
3. **Wrong server targeted** - PM2 server on port 3001 kept running; I kept trying to fix a server that wasn't broken

### Key Rules Going Forward
1. **Push to GitHub same day** - Never end a session without pushing
2. **Use explicit paths only** - No wildcards or patterns in file paths
3. **Check PM2 status first** - `pm2 list` before trying to restart anything
4. **Production mode is fine** - `npm start` is stable, use it
