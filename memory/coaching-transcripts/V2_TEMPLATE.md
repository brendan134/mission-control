# Coaching Call V2 Template

**Use this for all new transcript extractions from July 2025.**

## Required Fields (ALL 11 MUST BE PRESENT)

```
# Coaching Call - [Client Name]

**Date:** [Month DD, YYYY]                                    <- FIELD 1
**Duration:** ~[X] minutes                                   <- FIELD 2
**Call Type:** [Regular Coaching / Strategy Session / Impromptu / etc]  <- FIELD 3
**Energy/State:** [Warm, Direct, Reflective, etc]           <- FIELD 4
**Tags:** #[hashtag1] #[hashtag2] #[hashtag3]               <- FIELD 5

## Client
[Full Name/Title] - [Organization]                            <- FIELD 6

## Key Topics                                                  <- FIELD 7

### 1. [Main Topic]
- Detail point
- Another point

### 2. [Second Topic]
- Detail point

## Action Items                                                 <- FIELD 8
- [ ] Action item 1
- [ ] Action item 2

## Key Quotes                                                   <- FIELD 9
> "Quote from client or coach"
> "Another quote"

## Coaching Approach                                           <- FIELD 10
- How coach approached the conversation
- Specific frameworks or questions used
- What worked / what to adjust

## Content Potential                                           <- FIELD 11
- **Topic idea** - brief description
- Another potential content piece
```

**THE 11 FIELDS:**
1. Date | 2. Duration | 3. Call Type | 4. Energy/State | 5. Tags | 6. Client | 7. Key Topics | 8. Action Items | 9. Key Quotes | 10. Coaching Approach | 11. Content Potential

## V2 Format Checklist (DO NOT skip)

Before marking a transcript as "complete," verify:

- [ ] **Date** - in "Month DD, YYYY" format
- [ ] **Duration** - approximate minutes
- [ ] **Call Type** - category of call
- [ ] **Energy/State** - atmosphere/tone
- [ ] **Tags** - minimum 2-3 hashtags with #
- [ ] **Client** - name + organization
- [ ] **Key Topics** - numbered list with details
- [ ] **Action Items** - checkbox format [ ]
- [ ] **Key Quotes** - quoted text, minimum 2
- [ ] **Coaching Approach** - summary of coaching method
- [ ] **Content Potential** - at least 2 content ideas

## Anti-Patterns (DO NOT)

- ❌ Creating files in old format "first, convert later"
- ❌ Skipping tags section
- ❌ Using "Key Topics" without sub-details
- ❌ Forgetting Content Potential section
- ❌ Marking "done" without verification

## VALUE ADD SECTION (Required for all extractions)

This section provides DONE-FOR-YOU items to help the client take action.

### Done-For-You Recommendations (MINIMUM 3)

For each recommendation:
- **Recommendation Title:** [Descriptive title]
- **What it is:** [e.g., Email draft, Template, Checklist, Script]
- **The Content:** [Actual ready-to-use content]

Example:
```
### 1. Team Meeting Agenda - Initiative Focus
**What it is:** 30-minute team meeting agenda for Wednesday
**Content:**
- 5 min: Welcome & purpose
- 10 min: What does "proactive" look like?
- 10 min: Each person shares their May focus
- 5 min: Commitment + follow-up
```
