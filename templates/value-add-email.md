# Value-Add Email Template

## Subject Line Options

- **[Client First Name]**, here's that [tool/resource] from our call
- [Client First Name], quick follow-up from today
- [Client First Name] - [One-line summary of value-add]

## Email Body

Hey [Client First Name],

Hope you're doing well! 👋

As promised, here's [the [tool/resource name]] from our call:

[2-3 sentences describing the value-add and how to use it]

[If there's a specific action item:]
Why not try [specific action] this week and let me know how you go?

Looking forward to hearing about your progress!

Cheers  
Brendan

---

## Value-Add Types Reference

| Type | Example | Best For |
|------|---------|----------|
| Template | SWOT analysis, pricing calculator | Immediate implementation |
| Checklist | Staff review checklist, onboarding steps | Process clarity |
| Letter | WISE letter, formal notification | Legal/formal needs |
| Guide | 5-step process, quick start guide | Learning |
| Resource Link | External tool, video, article | Deepening |

## Variables

- `[Client First Name]` - Client's first name
- `[Client Full Name]` - Client's full name
- `[Value-Add Title]` - Name of the deliverable
- `[Value-Add Description]` - What it is and how to use it
- `[Specific Action]` - Recommended next step
