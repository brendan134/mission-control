# Mission Control - LBD AI System Dashboard

A clean, Linear-style interface for managing your AI system.

## Getting Started

```bash
cd mission-control
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.

## Features

- **Dashboard** - System overview, cost tracking, agent status
- **Agents** - View and manage specialist agents
- **Automations** - See scheduled cron jobs
- **Packs** - Quick access to retrieval packs
- **Tools** - Quick access to system tools

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Lucide Icons
