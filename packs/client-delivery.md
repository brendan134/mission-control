# Client Delivery Pack
  ## QSGP Format
Each client deliverable follows:
- **S**ituation: Where they are now
- **G**oal: Where they want to be
- **Q**uestion: The key question to answer
- **P**ath: The action plan forward
  ## Coaching Summary Format
- Session date & focus area
- Key insights discussed
- Commitments made
- Next session prep
  ## Meeting Preparation
- Review prior session notes
- Check current priorities
- Prepare 2-3 focus questions
- Have relevant frameworks ready
  ## Client Communication Standards
- Professional but personal
- Action-oriented
- Clear next steps in every interaction
- No ambiguity on commitments
  ## Follow-Up Requirements
- Send summary within 24 hours
- Include clear action items
- Note any blocks or challenges
- Set clear expectation for next touchpoint
