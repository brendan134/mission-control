# Frameworks Pack
  
## Core Business Frameworks
  
### The Accountability Framework
1. Define the outcome clearly
2. Assign ownership
3. Set check-in cadence
4. Measure and review
5. Adjust and improve
  
### Team Performance Framework
1. Clear role definition
2. Explicit expectations
3. Right systems/tools
4. Regular feedback loops
5. Consequence and recognition
  
### Scaling Architecture
1. Document the process
2. Systematize the process
3. Delegate to team
4. Oversee, don't overdo
5. Optimize continuously
  
### Self-Managing Team Indicators
- Team solves problems without owner
- Decisions made at appropriate level
- Consistent execution without supervision
- Owner focuses on strategy, not tactics
- Team owns outcomes
  
### The Owner Dependency Test
Can this task be done without me?
- Yes → delegate
- No → document for future delegation
- Maybe → start teaching someone
  
### Weekly Execution Rhythm
- Monday: Priorities set
- Wednesday: Check-in
- Friday: Review and celebrate
- Weekend: Rest and think
