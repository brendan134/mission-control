# Curriculum Pack
  ## Lesson Structure
- Clear outcome for the lesson
- Brief context/why it matters
- Core concept or framework
- Action step or implementation
- Wrap-up with next lesson connection
 
## Lesson Types
- **Concept lessons:** Explain a framework or principle
- **Implementation lessons:** Step-by-step execution
- **Case study lessons:** Real examples of the framework in action
- **Reflection lessons:** Prompt self-assessment
- **Workshop lessons:** Interactive exercises
 
## Worksheet Structure
1. A clear title
2. 3-5 questions or prompts
3. Space for answers/notes
4. Closing prompt for next step
 
## Masterclass Format
- Hook (problem + stakes) — 5 min
- Framework (core concept) — 15 min
- Application (examples) — 15 min
- Implementation (action steps) — 10 min
- CTA (next step) — 5 min
 
## Content Standards
- Always connect to team performance or operational improvement
- Include 1-2 action items per lesson
- No fluff — every minute should deliver value
- End with clear next step

## QSGP Format
- Situation: Current state
- Goal: Target state  
- Question: Key question to answer
- Path: Action plan forward
