## Common Standards

All content agents follow these v2.0 framework standards:

- **Approval Workflow**: draft → needs_revision → approved
- **Scoring**: 85 passing, 92 excellent
- **Voice**: Leader by Design (quiet authority)
- **Confidentiality**: Anonymised examples only

## Quick Start

1. Select content type (newsletter/linkedin/video/email)
2. Answer pre-write diagnostic questions
3. Draft content following format guidelines
4. Validate against scoring rubric
5. Submit for approval
