## Topic Selection Criteria

### Must Have
- [ ] Relates to business owners/leaders with teams
- [ ] Connects to leadership design framework
- [ ] Can be 10-13 minutes
- [ ] Fits one of the 5 formats

### Should Have
- [ ] Real example from coaching/transcripts
- [ ] Both objections naturally addressed
- [ ] Primary + secondary keywords available

### Nice to Have
- [ ] Topical angle (optional)
- [ ] Seasonal relevance (optional)

## Format Selection Guide

Based on Rotation Rules:

| If topic is... | Use Format |
|----------------|-----------|
| Leaders keep repeating the same mistake | Leadership Pattern |
| Common belief that needs challenging | Thought Reversal |
| Leaders feel understood, build trust | Connection |
| Clear solution/framework to teach | Educational |
| Client story with insight | Experimental/Story |

### Rotation Rules (LOCKED)
- Max 2 Educational back-to-back
- Connection every 3-4 weeks
- Story ~1x per quarter
- LP + TR never clustered

## Topical vs Evergreen Matrix

| Type | Description | Best For |
|------|-------------|----------|
| **Evergreen** | Fundamental leadership challenges | Most episodes (80%) |
| **Topical** | Tied to current events/season | Occasional (20%) |

### Evergreen Topics (Examples)
- "Why your team isn't performing" (always relevant)
- "The delegation trap"
- "Decision fatigue in leadership"

### Topical Topics (Examples)
- "Leadership lessons from [recent news]"
- "New year, new leadership habits"
- "Q4 perspective on team goals"

**Note:** Even topical episodes should work as evergreen - the topical hook brings them in, but the insight lasts forever.
