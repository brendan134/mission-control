## Production Workflow

### 1. Episode Intent (Pre-Write Diagnostic)

**Ruby MUST answer these 9 questions before writing:**

1. What false belief is this episode breaking?
2. Why does this belief persist for business owners and leaders?
3. What does the listener usually do instead?
4. What structural correction replaces that pattern?
5. Where will the primary objection surface naturally?
6. Where will the secondary objection surface naturally?
7. What is the cleanest searchable framing of this problem?
8. Which primary keyword best anchors this episode?
9. Which secondary keywords best support the episode without keyword stuffing?

### 2. Format Selection (Locked)

Must select ONE of these formats:

| Format | Purpose |
|--------|---------|
| **Leadership Pattern** | Identify recurring breakdowns, reframe through structural insight |
| **Thought Reversal** | Flip common belief, replace with design thinking |
| **Connection** | Link two leadership ideas, reveal relationships |
| **Educational** | Teach behaviour/decision process through explanation |
| **Experimental/Story** | Narrative-based insight with structure |

### 3. Script Writing

**Requirements:**
- 10-13 minutes spoken delivery (LOCKED)
- Embed both objections naturally in narrative
- Use quiet authority voice
- No motivational fluff or hype
- Follow 9-step structure:
  1. Hook (20-30 sec)
  2. Pattern
  3. Day-to-Day reality
  4. Why It Persists
  5. The Cost
  6. Reframe
  7. Reflection
  8. The Shift
  9. Close

### 4. Validation (Scoring Rubric)

**Must pass ALL of these:**

| Category | Weight | Passing Criteria |
|----------|--------|------------------|
| Structural Clarity | 15% | Moves from tension → insight → correction → standard |
| Voice Accuracy | 12% | Sounds like Brendan Rogers' Leader by Design |
| Leadership Depth | 12% | Beyond tactics into structural thinking |
| Objection Handling | 12% | Both objections surfaced & neutralised naturally |
| SEO Alignment | 12% | Keywords aligned across title, script, description |
| Title Quality | 8% | Clear, searchable, non-clickbait |
| Description Quality | 8% | Supports discoverability without promotion |
| Listener Retention | 8% | Opening creates tension, transitions sustain |
| Practical Relevance | 7% | Creates useful leadership shift |
| Originality | 6% | Not repetitive within format |

**Scoring:**
- 92+ → Excellent (publish-ready)
- 85-91 → Good (selective refinement)
- 84 or less → Needs revision

### 5. Description Creation

**5-Paragraph Locked Template:**

1. **Hook** - Introduce problem in searchable language
2. **Problem Reframing** - Challenge common assumption
3. **What This Episode Covers** - Clear listener outcome
4. **Keyword-Rich Value** - Primary + secondary keywords naturally
5. **Fixed Soft CTA** - "Listen to the full episode..."

---

## Output Standards

### Episode Package Must Include:

- [ ] Episode title (searchable, non-clickbait)
- [ ] 10-13 minute script
- [ ] Show notes / description (120-220 words)
- [ ] Primary keyword used
- [ ] 2+ secondary keywords
- [ ] Both objections neutralised
- [ ] CTA (soft, non-salesy)

### Tone Protection

**BLOCK these phrases:**
- "just", "simply"
- "all you need to do"
- "game changer"
- "massive breakthrough"
- "unlock your leadership"
- "high-performing leaders always"
- "crush it", "next-level", "dominate"

## Hard Fail Conditions

Episode is REJECTED if:

- Episode < 10 minutes or > 13 minutes
- Missing primary objection resolution
- Missing secondary objection resolution
- Format not locked/selected
- Tone drifts from Leader by Design
- Confidentiality breach
- Primary keyword missing
- Description template non-compliant
