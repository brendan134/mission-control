## Core Principle

**ALL YouTube Shorts content must be STANDALONE**

- Viewer should understand full message without needing the podcast
- No podcast dependency required to get value
- Podcast CTA goes in YouTube DESCRIPTION only - never in the video itself

---

## Content Structure

For each video, create:

### 1. Title Banner
- Top text shown in video
- Max 8 words
- Hook + curiosity

### 2. Caption Banner
- Bottom text shown in video
- Supporting line to title

### 3. Script
- 45-75 seconds
- Standalone message (no reference to podcast)
- Hook in first 3 seconds

### 4. YouTube SEO

#### Title
- Max 100 characters
- Include primary keyword
- No clickbait, all caps, or excessive punctuation

#### Description
- Min 150 words
- Structure: Hook → Value → Chapters → CTA → Hashtags

#### Tags
- Max 15 tags
- Priority: primary keyword, secondary, branded, related
