## Formatting Rules

| Rule | Requirement |
|------|-------------|
| **Emdashes** | NEVER USE. Use hyphens (-) instead. |
| **CTA (LinkedIn)** | "Listen to the full episode: link in comments below" |
| **CTA (Facebook)** | Same as LinkedIn: "Listen to the full episode: link in comments below" |
| **CTA (Instagram)** | "Listen to the full episode: link in comments below" or "Link in bio" |
| **Formatting** | Natural, conversational. No excessive bold/headers. Short sentences. |
| **Tone** | Calm, confident, not salesy. Never AI-structured. |
| **Hashtags** | 5 per platform per post |
